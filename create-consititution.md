**Constitution = Aapke project ke rules + system + zimmedariyan.**
Jaise family ke ghar ke rules hote hain, waise project ke rules likhne ko constitution kehte hain.

**Yeh 3 cheezein define karti hai:**

2. **Kon kya kaam karega?**
3. **Decisions kaise honge?**
